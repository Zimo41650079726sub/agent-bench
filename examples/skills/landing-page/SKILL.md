# landing-page

Generates a single-file landing page for a product.

## Requirements

Write the page to `/workspace/output/index.html` as **one self-contained
HTML file** (all CSS inline in a `<style>` block, no external resources,
no JavaScript required).

The page must contain, in order:

1. A hero section with the product name as the main heading and a one-line
   tagline.
2. A features section with exactly **three** feature cards, each with a
   short title and one sentence.
3. A footer with a copyright line.

## Steps

1. Read this file.
2. Write the complete page to `/workspace/output/index.html`.
3. Verify the file exists (e.g. `ls output/`).
